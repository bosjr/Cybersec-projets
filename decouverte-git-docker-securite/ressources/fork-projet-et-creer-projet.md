# Procédure étudiant — Fork du dépôt formateur + création du projet personnel

## Objectif

Vous allez :

1. créer un fork du dépôt enseignant
2. cloner ce fork avec GitHub Desktop
3. créer votre propre projet personnel dans :

```text
Documents\GitHub\Monprojet
```

---

## Partie 1 — Fork du dépôt enseignant

Dépôt enseignant :

[Cybersec-projets](https://github.com/bosjr/Cybersec-projets)

---

## Étape 1 — Ouvrir le dépôt

Ouvrir :

[Cybersec-projets](https://github.com/bosjr/Cybersec-projets)

---

## Étape 2 — Créer le fork

En haut à droite :

```text
Fork
```

Puis :

```text
Create fork
```

GitHub crée alors :

```text
https://github.com/VOTRE_LOGIN/Cybersec-projets
```

Le dépôt apparaît maintenant sur votre compte GitHub.

---

## Partie 2 — Cloner le fork avec GitHub Desktop

### Étape 3 — Ouvrir GitHub Desktop

## Étape 4 — Cloner le dépôt

Menu :

```text id=
File → Clone Repository
```

---

## Étape 5 — Sélectionner le fork Cybersec-projets

Onglet :

```text
GitHub.com
```

Sélectionner :

```text
VOTRE_LOGIN/Cybersec-projets
```

---

## Étape 6 — Choisir le dossier local

Dans :

```text
Local Path
```

choisir :

```text
Documents\GitHub
```

GitHub Desktop créera automatiquement :

```text
Documents\GitHub\Cybersec-projets
```

---

## Étape 7 — Cloner

Cliquer :

```text
Clone
```

Le dépôt est maintenant disponible localement.

---
---

## Partie 3 — Créer votre projet personnel

### Étape 8 — Nouveau dépôt personnel

Dans GitHub Desktop :

```text
File → New Repository
```

---

## Étape 9 — Paramètres

### Repository Name

```text
Monprojet
```

---

## Local Path

Choisir :

```text
Documents\GitHub
```

GitHub Desktop créera automatiquement :

```text
Documents\GitHub\Monprojet
```

---

## Cocher

```text
Initialize this repository with a README
```

---

## Créer le dépôt

Cliquer :

```text
Create Repository
```

---

## Partie 4 — Publier le projet personnel sur GitHub

## Étape 10 — Publication

Cliquer :

```text
Publish Repository
```

---

## Étape 11 — Vérifier

Le dépôt personnel apparaît sur votre compte GitHub :

```text
https://github.com/VOTRE_LOGIN/Monprojet
```

---
