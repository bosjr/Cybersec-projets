# Projet Découverte Git, Docker et Sécurité

## 🎯 Objectif

Ce projet a pour objectif de vous faire découvrir les bases de :

- Git et la gestion de version
- GitHub et le travail collaboratif
- Docker et le déploiement d’applications
- Les vulnérabilités web

Ce projet est destiné à des étudiants débutants (Bachelor 1).

---

## 🧭 Organisation

Le projet est structuré en plusieurs étapes :

0. Installation des outils VsCode GitHub Studio 2H
1. Prise en main de GitHub 2H
2. Utilisation de Git 2H
3. Lancement d’une application avec Docker 2H
4. Découverte d’une vulnérabilité 2H
5. Restitution du travail 2H
